# 城市宣传海报 · 新疆

**[English](README_EN.md)**

2026 新疆城市宣传海报 AI 提示词与生成图集合，涵盖新疆 42 个城市与县域。

## 项目简介

本项目为新疆维吾尔自治区 42 个城市/县量身定制了 **国潮风格** 城市宣传海报。每个城市包含：

- **`prompt.md`** — AI 绘图提示词，可用于 Midjourney、Stable Diffusion、DALL·E 等工具生成海报
- **`城市名.png`** — 基于提示词生成的 9:16 竖版宣传海报

## 设计语言

所有海报采用统一的视觉风格：

- **双重曝光** — 人物与城市景观的融合
- **S 型流动构图** — 绸带飘飞延展为城市山河
- **国潮手绘长卷** — 融入城市地标与自然风光
- **民族服饰** — 维吾尔族、哈萨克族、回族、蒙古族等传统服饰
- **大面积留白** — 清透飘逸、仙气十足
- **城市专属配色** — 每座城市拥有独特的色彩方案

## 使用方法

1. 进入目标城市目录，打开 `prompt.md` 获取提示词
2. 将提示词复制到 AI 绘图工具（推荐 Midjourney、Stable Diffusion 等）
3. 生成 9:16 竖版海报
4. 可根据需要微调提示词中的配色、地标等元素
